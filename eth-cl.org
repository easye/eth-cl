* Installation
With an ANSI Common Lisp implementation configured with Quicklisp
#+caption: Needed once
#+begin_src lisp :results raw
  (ql:quickload "eth-cl")
  (asdf:make :eth-cl)
#+end_src

* Usage

#+caption: Create a new Ethereum account
#+begin_src lisp 
  (eth:make-account)
#+end_src
